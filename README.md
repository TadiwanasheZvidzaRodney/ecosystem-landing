# Stratus - The Infrastructure for African Innovation

Enterprise consulting, open data, and community-driven innovation. Three pillars powering the next generation of African technology.

## Tech Stack

- **Framework:** Nuxt.js 4
- **Styling:** Tailwind CSS
- **Icons:** Lucide Vue
- **Deployment:** Vercel

## Setup

Make sure to install dependencies:

```bash
npm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
npm run dev
```

## Production

Build the application for production:

```bash
npm run build
```

Preview production build locally:

```bash
npm run preview
```

## Deploy

Deploy to Vercel:

```bash
vercel
```
